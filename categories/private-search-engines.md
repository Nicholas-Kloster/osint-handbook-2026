# Private Search Engines

> 19 tools — [osinthandbook.com/private-search-engines](https://www.osinthandbook.com/private-search-engines)

- [Brave Search](https://search.brave.com/) — Privacy-focused search engine developed by Brave, offering anonymous browsing and integrated ad-blocking.
- [DuckDuckGo](https://duckduckgo.com) — Search engine that prioritises privacy and does not track users or store personal data.
- [etools.ch](https://www.etools.ch) — Swiss metasearch engine that aggregates results from various major search engines.
- [Gibiru](https://gibiru.com) — Uncensored search engine claiming to deliver unfiltered and private search results.
- [Kagi](https://kagi.com/) — Subscription-based search engine focused on high-quality, ad-free and privacy-respecting search.
- [Lukol](https://www.lukol.com) — Proxy-based search engine that anonymises queries while using Google search results.
- [MetaGer](https://metager.de) — Open-source German metasearch engine emphasising privacy and data security.
- [Mojeek\n](https://www.mojeek.com) — Independent crawler-based search engine that builds its own index with privacy by design.
- [My Private Search](https://www.myprivatesearch.com) — Privacy-oriented search engine designed to prevent tracking and profiling.
- [peekier](https://peekier.com) — Privacy-centric search engine showing site previews while avoiding personal data collection.
- [QWANT](https://www.qwant.com) — French search engine that respects user privacy and avoids tracking and personalisation.
- [Search encrypt](https://www.searchencrypt.com) — Search engine that encrypts search terms locally before sending them to its servers.
- [Seek Ninja](https://seek.ninja) — A privacy-focused search engine with its own crawlers and index.
- [Snap Search](https://snapsearch.online) — Privacy-respecting browser-as-a-service that renders websites anonymously.
- [Startpage](https://www.startpage.com) — Dutch search engine that serves Google results with strong privacy protections.
- [Stract](https://stract.com/) — Search engine that offers real-time web and social media search with emphasis on relevance.
- [Swisscows](https://swisscows.com) — Family-friendly Swiss search engine with built-in semantic search and strong privacy features.
- [Whoogle-search](https://github.com/benbusby/whoogle-search) — Self-hosted, open-source proxy for Google Search with no tracking or ads.
- [Yep](https://yep.com/) — Search engine developed by Ahrefs that shares ad revenue with content creators.
