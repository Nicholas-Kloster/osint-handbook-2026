# Meta Search

> 19 tools — [osinthandbook.com/meta-search](https://www.osinthandbook.com/meta-search)

- [100SearchEngines](https://www.100searchengines.com) — Directory listing numerous search engines for various topics and regions.
- [All-in-One](https://all-io.net/) — Metasearch engine that consolidates multiple search engines into one interface.
- [AllTheInternet](https://www.alltheinternet.com/) — Search engine providing access to multiple search tools and resources.
- [Dogpile](http://www.dogpile.com) — Metasearch engine that compiles results from various search engines like Google and Yahoo.
- [Etools](https://www.etools.ch/) — Swiss-based metasearch engine focusing on privacy and transparency.
- [Excite](https://www.excite.com) — Web portal offering a variety of services including search, email, and news.
- [FaganFinder](https://www.faganfinder.com) — Website offering various search tools and resources for research purposes.
- [falla](https://github.com/Sanix-Darker/falla) — Command-line scraper supporting over 15 search engines, including Google and Bing, designed for educational purposes to perform automated searches.
- [Findwith.me](https://www.findwith.me/) — Search engine that allows users to search across various platforms simultaneously.
- [Info.com](http://www.info.com) — Search engine that presents results from multiple search engines in one place.
- [Infospace](http://www.infospace.com) — A metasearch engine that aggregates results from multiple sources like Google, Yahoo, and Bing.
- [iZito](https://www.izito.com/) — Metasearch engine delivering aggregated search results from various sources.
- [Query Tool](https://github.com/oryon-osint/querytool) — Open-source OSINT framework using Google Sheets for advanced search queries.
- [Search-it](https://search-it.netlify.app) — Web application that allows users to search using multiple search engines simultaneously.
- [SearchAll](https://www.searchall.net) — Metasearch engine that compiles results from multiple search engines.
- [The Lookup](https://the-lookup.com/) — Website offering easy access to multiple search tools.
- [WebCrawler](http://www.webcrawler.com) — One of the oldest metasearch engines, combining results from various search engines.
- [Yubnub](https://yubnub.org/) — Command-line-style search engine allowing users to create and use custom search commands.
- [Zapmeta](https://www.zapmeta.com/) — Metasearch engine providing combined results from several search engines.
