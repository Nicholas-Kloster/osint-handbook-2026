# Document Metadata

> 21 tools — [osinthandbook.com/document-metadata](https://www.osinthandbook.com/document-metadata)

- [_intelligenceX Magic File Tool](https://intelx.io/tools?tab=filetool) — Tool for extracting and analysing text, metadata, selectors and images from supported files.
- [Anonymouth](https://github.com/psal/anonymouth) — A Java-based application that aims to give users tools and knowledge needed to begin anonymising documents they have written.
- [Crossref](https://search.crossref.org/) — Search the metadata of journal articles, books, standards, datasets, and more.
- [DeepFind.me: Metadata Extractor](https://www.deepfind.me/tools/file-analysis/metadata-extractor) — Upload a file to extract its metadata. Supports images, videos, audio files, and documents.
- [Doc Scrubber](http://www.brightfort.com/docscrubber.html) — Windows utility to remove hidden metadata from Word documents.
- [Evermap AutoMetadata](https://evermap.com/autometadata.asp) — Tool to automatically update or manage metadata in PDF files.
- [ExtractMetadata](https://www.extractmetadata.com/) — Online tool to extract metadata from documents, images, and other file types.
- [FOCA](https://github.com/ElevenPaths/FOCA) — Open-source version of FOCA for metadata and document analysis.
- [Goblyn](https://github.com/loseys/Goblyn) — A Python tool focused to enumeration and capture of website files metadata.
- [Goca](https://github.com/gocaio/goca) — A FOCA fork written in Go.
- [GroupDocs](https://products.groupdocs.app/metadata/total) — Online app to view, edit, and clean metadata across multiple file types.
- [Mat2](https://0xacab.org/jvoisin/mat2) — A metadata removal tool, supporting a wide range of commonly used file formats.
- [Metadata2Go](https://www.metadata2go.com) — Free service to read metadata from various file formats directly in the browser.
- [Metagoofil](http://www.edge-security.com/metagoofil.php) — Tool for extracting metadata from public documents found through search engines.
- [Metashield Clean-up-Online](https://metashieldclean-up.elevenpaths.com) — Tool by ElevenPaths to analyse and clean metadata from files before sharing.
- [PDF Candy Edit PDF Metadata](https://pdfcandy.com/edit-pdf-meta.html) — Edit or remove PDF metadata free of charge.
- [PDF Yeah Read PDF Metadata](https://www.pdfyeah.com/view-pdf-metadata) — Web-based service to view metadata from uploaded PDF documents.
- [Pymeta](https://github.com/m8r0wn/pymeta) — Utility to download and extract document metadata from an organisation.
- [SEAL](https://cybersecuritycloud.telefonicatech.com/innovacion-labs/tecnologias-innovacion/foca) — FOCA is a metadata and network reconnaissance tool used for footprinting documents.
- [Spiderpig](https://github.com/hatlord/Spiderpig) — A document metadata harvester.
- [xeuledoc](https://github.com/Malfrats/xeuledoc) — A script that fetches information about a public Google document.
