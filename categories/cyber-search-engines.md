# Cyber Search Engines

> 19 tools — [osinthandbook.com/cyber-search-engines](https://www.osinthandbook.com/cyber-search-engines)

- [Binary Edge](https://www.binaryedge.io) — Cybersecurity search engine indexing internet-facing devices and services.
- [Censys](https://censys.io) — Search engine and data platform for discovering internet-connected assets.
- [Censys Workshop](https://workshop.censys.io) — A collection of prototypes and beta projects from the team at Censys including CensysGPT, Map to Censys, and Threatfox Censys.
- [Cybersecurity Search Engine](https://www.criminalip.io/) — Threat intelligence platform for scanning and monitoring cyber risks.
- [FOFA](https://fofa.so) — Cyber asset search engine widely used in China for network reconnaissance.
- [Full Hunt\n](https://fullhunt.io/) — Attack surface management and reconnaissance platform for enterprises.
- [Grey Noise](https://viz.greynoise.io) — A cybersecurity platform that collects and analyses Internet-wide scan and attack data.
- [Hackyx](https://hackyx.io) — Search engine for hackers.
- [Hunter](https://hunter.how/) — Search engine for internet-exposed assets and technologies.
- [LeakIX](https://leakix.net) — Searchable index of exposed misconfigurations leading to leaks and security issues.
- [Netlas](https://netlas.io/) — Search engine for internet metadata and infrastructure.
- [ODIN](https://odin.io/) — Search engine and data platform for discovering internet-connected assets.
- [Onyphe](https://www.onyphe.io) — Cyber defence search engine aggregating threat intelligence and internet data.
- [Quake](https://quake.360.net/) — Chinese internet asset search engine.
- [Shodan](https://www.shodan.io/) — Search engine for internet-connected devices and open ports.
- [Shodan Filters](https://github.com/T43cr0wl3r/shodan-filters) — Collection of advanced Shodan search filters for reconnaissance.
- [Shodan Scripts](https://github.com/random-robbie/My-Shodan-Scripts) — Scripts leveraging Shodan for reconnaissance and automation.
- [ShodanSpider](https://github.com/shubhamrooter/ShodanSpider) — Python-based tool to crawl data from Shodan using API.
- [ZoomEye](https://www.zoomeye.org) — Chinese search engine for cyber asset discovery and intelligence.
