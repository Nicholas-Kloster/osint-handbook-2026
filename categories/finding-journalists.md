# Finding Journalists

> 11 tools — [osinthandbook.com/finding-journalists](https://www.osinthandbook.com/finding-journalists)

- [A News Tip](https://anewstip.com/) — Searchable directory of media contacts.
- [Muck Rack](https://muckrack.com) — Software for managing PR workflows, including media monitoring, journalist outreach, and reporting.
- [notified](https://www.notified.com) — Communications platform for PR, IR, and ESG professionals to distribute news and analyse impact.
- [OnePitch](https://onepitch.co) — Platform matching journalists' needs with expert sources through curated pitching.
- [Press Hunt](https://presshunt.co) — Platform connecting journalists and PR professionals to share newsworthy pitches.
- [PressCloud](https://presscloud.ai) — PR software automating press release creation, distribution, and media tracking.
- [PressPage](https://presspage.com) — Press release management software with multimedia newsroom, analytics, and distribution tools.
- [Prowly](https://prowly.com) — System supporting press release creation, media outreach, coverage tracking, and performance reporting.
- [qwoted](https://www.qwoted.com) — Marketplace that helps journalists find expert sources and story contributors.
- [Response Source](https://www.responsesource.com) — UK-based PR network enabling experts to respond to journalist requests.
- [Roxhill Media](https://roxhillmedia.com) — Service offering media outreach tools and curated journalist contacts database.
