# Investigation Tracking

> 11 tools — [osinthandbook.com/investigation-tracking](https://www.osinthandbook.com/investigation-tracking)

- [Altia's OSINT Investigator](https://altiaintel.com/our-products/osint-investigator) — A comprehensive solution allowing investigators capture online content.
- [Forensic Notes](https://www.forensicnotes.com) — Secure digital notebook for investigators that timestamps, hashes, and encrypts notes to ensure evidential integrity.
- [Forensic OSINT](https://www.forensicosint.com/) — Browser-based OSINT capture tool that records webpages, videos, and social media content with automatic timestamps and hashes.
- [Hunchly](https://www.hunch.ly) — Automated web capture tool for investigators that logs every visited page with metadata, tags, and case management features.
- [OSIRT Browser](https://osirtbrowser.com/) — Desktop web investigation tool that captures and logs online evidence into a local, timestamped casebook for forensic reporting.
- [Pagefreezer](https://www.pagefreezer.com) — A tool for fully automated, compliant archiving of websites and social media for legal, audit, and eDiscovery needs.
- [Paliscope](https://www.paliscope.com) — An AI-powered open source investigation platform that helps analysts search, organise, analyse, and document data.
- [Save the Proof](https://www.savetheproof.com/) — A web-based service for legally certifying digital evidence (webpages, browsing sessions, files, chats) with compliant digital signatures and timestamps.
- [Single File](https://github.com/gildas-lormeau/SingleFile) — Web Extension for saving a faithful copy of a complete web page in a single HTML file.
- [Ubikron](https://www.ubikron.com) — Ubikron keeps track of your web research by storing, tagging, and enriching the web visits and screen clippings.
- [Vortimo](https://www.osint-tool.com) — A browser extension (and supporting web application) that offers a real-time, on-page approach to analysing web content.
