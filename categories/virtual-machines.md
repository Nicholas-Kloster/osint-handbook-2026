# Virtual Machines

> 9 tools — [osinthandbook.com/virtual-machines](https://www.osinthandbook.com/virtual-machines)

- [Browser Appliance](http://browser.shell.tor.hu) — Secure VirtualBox-based VM for safe and isolated web browsing.
- [Linux KVM](https://www.linux-kvm.org/page/Main_Page) — Main information and documentation hub for KVM, Linux’s built-in hypervisor.
- [Parallels](https://www.parallels.com) — Virtualisation solution for Mac offering seamless Windows and multi-OS support.
- [QEMU](https://www.qemu.org) — Cross-platform open-source machine emulator and virtualiser.
- [Trace Labs OSINT VM](https://www.tracelabs.org/trace-labs-osint-vm) — A curated virtual machine packed with OSINT tools for investigations.
- [Veertu](https://veertu.com) — macOS/iOS CI/CD virtualisation platform providing ephemeral macOS VMs and CVE scanning for DevOps workflows.
- [VirtualBox](https://www.virtualbox.org) — A general-purpose full virtualisation software for x86_64 hardware (with version 7.1 additionally for macOS/Arm and with version 7.2 also for Windows/Arm), targeted at laptop, desktop, server and embedded use.
- [VMWare](https://www.vmware.com) — Comprehensive suite of virtualisation and cloud infrastructure products for desktop and enterprise.
- [Windows Hyper-V](https://www.microsoft.com/en-us/cloud-platform/server-virtualization) — Microsoft’s official page for server virtualisation offerings.
