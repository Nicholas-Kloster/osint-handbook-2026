# Vpn Testing

> 7 tools — [osinthandbook.com/vpn-testing](https://www.osinthandbook.com/vpn-testing)

- [BrowserLeaks: WebRTC Leak Test](https://browserleaks.com/webrtc) — Detects WebRTC IP leaks in browsers.
- [Check IP](https://www.perfect-privacy.com/check-ip) — IP address and DNS leak testing tool.
- [DNS Leak](https://www.dnsleaktest.com) — Comprehensive DNS leak testing tool.
- [DNS Leak Test](https://dnsleak.com/) — Tests for DNS leaks and privacy issues.
- [Email Leak Test](https://emailipleak.com/) — Checks email services for IP address leaks.
- [IPCheck](http://ip-check.info) — Checks your public IP address and connection details.
- [IPv6 Leak Test](https://ipv6leak.com/) — Tests for IPv6 leaks and privacy vulnerabilities.
