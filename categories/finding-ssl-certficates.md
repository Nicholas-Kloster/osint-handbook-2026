# Finding Ssl Certficates

> 14 tools — [osinthandbook.com/finding-ssl-certficates](https://www.osinthandbook.com/finding-ssl-certficates)

- [CERT Spotter](https://sslmate.com/certspotter/api) — API for monitoring Certificate Transparency logs for newly issued certificates.
- [Certdomainfinder](https://github.com/amar-myana/certdomainfinder/blob/master/README.md) — Tool for finding subdomains from Certificate Transparency logs.
- [Certgraph](https://github.com/lanrat/certgraph) — Tool that builds a graph of certificate relationships from Certificate Transparency data.
- [Certificate Details](https://certificatedetails.com/) — Website for searching and exploring SSL certificate information.
- [Certificate Search](https://crt.sh) — Search engine for Certificate Transparency logs.
- [certs.io](https://certs.io/) — Search tool for querying Certificate Transparency logs and certificate metadata.
- [DNS Lookup Pro](https://dnslookup.pro/) — Advanced online DNS record lookup service with detailed output.
- [Entrust Certificate Search](https://ui.ctsearch.entrust.com/ui/ctsearchui) — Tool for searching Certificate Transparency logs.
- [Internet-Wide Scan Data Repository](https://scans.io) — Archive of large-scale internet scan datasets, including certificate data.
- [Meta Certificate Transparency Monitor](https://developers.facebook.com/tools/ct/search/) — Facebook tool for searching Certificate Transparency logs.
- [OpenData Rapid7](https://opendata.rapid7.com) — Open data repository by Rapid7 offering cybersecurity scan data and research datasets.
- [OSINT.sh: Certificate Search](https://osint.sh/crt) — A tool from OSINT.sh to find information about the target assets from their SSL certificate.
- [Synapsint](https://synapsint.com) — A free service that aggregates data from multiple intelligence sources, search engines, datasets, and other resources to provide comprehensive cybersecurity insights.
- [tls.bufferover.run](https://tls.bufferover.run/) — Fast domain name lookups for TLS certificates in IPv4 space.
