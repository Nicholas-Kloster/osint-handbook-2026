# Identifying Backlinks

> 14 tools — [osinthandbook.com/identifying-backlinks](https://www.osinthandbook.com/identifying-backlinks)

- [Ahrefs Site Explorer](https://ahrefs.com/site-explorer) — Tool for analysing backlinks, organic traffic, and referring domains.
- [Backlink Watch](http://www.backlinkwatch.com) — Tool for checking backlinks to a given website.
- [CrawlCenter: External Links Checker](https://www.crawlcenter.com/external-link-checker) — Checker for analysing external links on a website.
- [Link Assistant Backlink Checker](https://www.link-assistant.com/seo-spyglass/free-backlink-checker-tool.html) — A free backlink checker.
- [Link Miner](https://linkminer.com) — Backlink analysis tool for SEO research.
- [Linkody Backlink Checker](https://www.linkody.com/en/seo-tools/free-backlink-checker) — Free backlink checker with SEO metrics and monitoring.
- [Majestic](https://majestic.com) — Tool for analysing backlinks and domain authority metrics.
- [MOZ Explorer](https://moz.com/explorer) — SEO tool for link analysis and keyword research.
- [MOZ Link Explorer](https://moz.com/link-explorer) — Advanced link analysis tool for checking domain and page-level backlinks.
- [OpenLinkProfiler](https://www.openlinkprofiler.org) — Free backlink analysis tool with live link updates.
- [SEO Tools: Free Backlink Checker](https://www.seoreviewtools.com/valuable-backlinks-checker/) — Tool for checking valuable backlinks and SEO metrics.
- [Small SEO Tools: Backlink Checker](https://smallseotools.com/backlink-checker/) — Free SEO tool for checking backlinks to a website.
- [The Hoth](https://www.thehoth.com/backlinks-checker/) — Backlink analysis tool with SEO metrics and competitor comparison.
- [WhoLinks2Me](http://www.wholinks2me.com) — Tool for finding inbound links and keywords information about a website.
