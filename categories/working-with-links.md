# Working With Links

> 12 tools — [osinthandbook.com/working-with-links](https://www.osinthandbook.com/working-with-links)

- [Bulk DA Checker: URL Extractor](https://www.bulkdachecker.com/url-extractor) — Tool that extracts URLs from large blocks of text or documents.
- [Bulk URL Opener Extension](https://chromewebstore.google.com/detail/bulk-url-opener-extension/hgenngnjgfkdggambccohomebieocekm) — Chrome extension for opening multiple URLs quickly in bulk.
- [CheckShortURL](http://checkshorturl.com) — Expand and verify a shortened link.
- [DeepFind.me: URL Unshortener](https://www.deepfind.me/tools/search-and-discovery/unshorten-url) — See the final destination of any shortened link before you click.
- [Expand URL](https://www.expandurl.net) — Expand and verify a shortened link.
- [Open All URLs](https://www.openallurls.com/) — Tool to open or test multiple URLs at once in different browser tabs.
- [Open Multiple URLs](https://chromewebstore.google.com/detail/open-multiple-urls/oifijhaokejakekmnjmphonojcfkpbbh) — Chrome extension that opens a list of URLs in new tabs simultaneously.
- [Redirect Detective](https://redirectdetective.com) — Service for tracing URL redirects to reveal final destination and intermediate steps.
- [Tiny Scan](https://www.tiny-scan.com/) — Mobile scanning app that turns documents and photos into high-quality PDFs.
- [unfurl](https://github.com/obsidianforensics/unfurl) — Open-source tool that parses URLs and extracts useful information from their structure.
- [Unshorten](https://unshorten.it) — Expand and verify a shortened link.
- [Where Does This Link Go?](http://wheredoesthislinkgo.com) — A URL redirect checker.
