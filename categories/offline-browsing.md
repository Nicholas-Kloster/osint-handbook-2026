# Offline Browsing

> 10 tools — [osinthandbook.com/offline-browsing](https://www.osinthandbook.com/offline-browsing)

- [A1 Website Download](http://www.microsystools.com/products/website-download/) — Tool for downloading entire websites to disk for offline browsing, converting dynamic sites to static, preserving link structure.
- [ArchiveBox](https://archivebox.io) — A powerful, self-hosted internet archiving solution to collect, save, and view websites offline.
- [Cyotek WebCopy](https://www.cyotek.com/cyotek-webcopy) — Free utility to copy websites locally for offline viewing.
- [HTTrack](http://www.httrack.com) — Website copier and offline browser for downloading entire websites to view locally.
- [Kiwix](https://www.kiwix.org) — Open-source offline browser for accessing Wikipedia and other educational content without internet connection.
- [Resolver](https://metaproductsrevolver.com) — Forensic tool suite for capturing websites, including snapshots, downloading large sites, useful in evidence collection.
- [SingleFile](https://github.com/gildas-lormeau/SingleFile) — Web Extension for saving a faithful copy of a complete web page in a single HTML file.
- [SiteSucker](https://ricks-apps.com/osx/sitesucker/) — macOS application that downloads entire websites for offline browsing.
- [Web Assistant](https://proxy-offline-browser.com/Download-MM3-WebAssistant-Proxy-Offline-Browser-Professional.html) — Proxy-based offline browser that archives pages visited.
- [Website Ripper Copier](https://www.tensons.com/products/websiterippercopier/) — Multi-threaded website downloader / ripper for offline browsing.
