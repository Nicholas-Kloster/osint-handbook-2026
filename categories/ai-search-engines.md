# Ai Search Engines

> 15 tools — [osinthandbook.com/ai-search-engines](https://www.osinthandbook.com/ai-search-engines)

- [Andi](https://andisearch.com) — A search engine employing generative AI to offer conversational answers.
- [Bagoodex](https://bagoodex.io) — An AI chatbot and search engine with a simple, clean interface.
- [ChatGPT](https://chatgpt.com/) — An LLM chatbot developed by OpenAI and released in late 2022.
- [Claude](https://claude.ai) — Anthropic’s advanced AI chatbot.
- [Consensus](https://consensus.app) — A search engine that uses AI to find answers in scientific research.
- [Felo](https://felo.ai) — An innovative AI-based search engine developed by the Japanese startup Felo Inc.
- [Gemini](https://gemini.google.com/) — Google's AI assistant.
- [Genspark](https://www.genspark.ai) — An AI-powered search engine that generates custom summaries called Sparkpages.
- [Komo AI](https://komo.ai) — An AI-powered search engine and subject-exploring tool.
- [Microsoft Copilot](https://copilot.microsoft.com) — Microsoft's AI assistant.
- [Perplexity AI](https://www.perplexity.ai) — A free, feature-rich, AI-powered answer search engine that retrieves information from multiple sources.
- [Phind](https://www.phind.com) — An AI-powered search engine that writes rich, visual, and interactive answers to the users' questions.
- [ThinkAny](https://thinkany.ai) — An advanced AI-powered search engine bringing together quality content retrieval and innovative AI-driven answering capabilities.
- [Waldo](https://www.waldo.fyi) — An AI-powered strategy engine.
- [You.com](https://you.com) — An AI assistant that began as a personalisation-focused search engine.
