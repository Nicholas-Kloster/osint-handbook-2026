# Browser Tests

> 27 tools — [osinthandbook.com/browser-tests](https://www.osinthandbook.com/browser-tests)

- [Am I Unique?](https://amiunique.org) — Website that analyses browser fingerprinting and uniqueness.
- [Browser Capabilities Project: User Agent Lookup](http://browscap.org/ua-lookup) — User-agent lookup and browser capability database
- [BrowserAudit](https://browseraudit.com) — Online tool to test browser privacy and security
- [Browserleaks](https://browserleaks.com) — Comprehensive browser security and privacy testing tools
- [CentralOps.net](https://centralops.net) — Suite of network and domain investigation tools including DNS, Whois, traceroute, and email verification.
- [Cover Your Tracks EFF](https://coveryourtracks.eff.org/) — A platform to test browser tracking and fingerprinting resistance.
- [Detect My Browser](https://detectmybrowser.com) — A website that displays detailed information about your browser.
- [Device Info](https://www.deviceinfo.me) — Provides detailed device and browser information.
- [Doileak](https://www.doileak.com) — Checks browser privacy leaks and fingerprinting.
- [IP Quality Score](https://www.ipqualityscore.com/) — Delivers advanced fraud detection and cybersecurity solutions, utilizing real-time risk scoring, proxy and VPN detection, email and phone validation, and device fingerprinting to identify and prevent malicious activities.
- [IPFingerprints](https://www.ipfingerprints.com/) — A set of tools for IP analysis.
- [IPLeak](https://ipleak.net) — Checks IP address leaks and online privacy.
- [My IP](https://myip.ms/) — IP intelligence platform with WHOIS, blacklist, and hosting data.
- [Netperf Tools](https://netperf.tools) — Network performance testing and monitoring tools.
- [Privacy.net: Privacy Analyzer](https://privacy.net/analyzer) — Online privacy and security analyser.
- [Qualys Browser Check](https://browsercheck.qualys.com) — Browser security and SSL / TLS testing tool.
- [Social Media Leak](https://robinlinus.github.io/socialmedia-leak/) — Detects social media platforms you are currently logged into.
- [Sploit.io](https://sploit.io) — A free service dedicated to finding and alerting to flaws and vulnerabilities present in the user's browser.
- [User Agent String](https://useragentstring.com/) — A tool to parse and analyse user-agent strings.
- [Web Okay](https://webkay.robinlinus.com/) — Comprehensive browser and device information tool.
- [What browser am I using?](http://www.whatbrowseramiusing.co) — Displays browser and system details.
- [What Is My Browser](https://explore.whatismybrowser.com/useragents/parse/#google_vignette) — Parses and analyses user-agent strings.
- [WhatIsMyBrowser](https://www.whatismybrowser.com) — Browser detection and user-agent information platform.
- [WhatIsMyIP](https://www.whatismyip.com) — Tool to instantly view your public IPv4/IPv6 address, ISP and location.
- [WhatIsMyIPAddress](http://whatismyipaddress.com) — Service showing your public IP (IPv4/IPv6), geolocation, ISP and privacy/security context.
- [WhatMyIP.co](http://whatmyip.co) — Simple site to display your public IP and basic network diagnostics (WHOIS, traceroute).
- [WHOER](https://whoer.net) — Find and check IP address.
