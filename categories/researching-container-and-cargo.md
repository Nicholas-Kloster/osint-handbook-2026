# Researching Container And Cargo

> 18 tools — [osinthandbook.com/researching-container-and-cargo](https://www.osinthandbook.com/researching-container-and-cargo)

- [Air Cargo Tracking](https://www.utopiax.org/) — Provides air cargo tracking and news.
- [BIC Code](https://www.bic-code.org) — Offers a registry for container prefixes and facility codes to standardize global intermodal transport.
- [Cargo Loop](https://www.cargoloop.com/) — Specializes in international vehicle shipping and offers access to U.S. auto shipping data.
- [Container Check Digit Calculator](https://container.checkdigit.info) — Calculates and verifies ISO 6346 container check digits for accurate container identification.
- [Container Tracking](http://container-tracking.org) — Aggregates container tracking data from multiple carriers for comprehensive shipment visibility.
- [ESA Package and Container Tracking](https://gssc.esa.int/navipedia/index.php/Package_and_Container_Tracking) — Explores GNSS-based tracking technologies for real-time monitoring of packages and containers.
- [FleetMon](https://www.fleetmon.com/services/container-tracker/search) — Enables users to track containers globally by integrating data from various shipping lines.
- [ImportGenius](https://www.importgenius.com) — Provides access to global trade data, helping businesses analyze import/export activities and supply chains.
- [ImportYeti\n](https://www.importyeti.com/) — Allows users to search U.S. customs records to identify suppliers and monitor competitors' shipments.
- [Maersk Tracking](https://www.maersk.com/tracking/#tracking) — Offers real-time tracking of containers and shipments across various transportation modes.
- [Panjiva Search](https://panjiva.com/search) — Provides detailed information on global suppliers and manufacturers through trade data analysis.
- [PIERS](https://ihsmarkit.com/products/piers.html) — Delivers U.S. import and export data, offering insights into trade activities and market trends.
- [Prefix Check Digit](https://www.prefixlist.com/codes/check-digit/) — Assists in calculating and validating container check digits to ensure compliance with ISO standards.
- [Prefix ISO Codes](https://www.prefixlist.com/codes/iso-codes/) — Offers a database of ISO codes for countries and currencies used in international trade.
- [Searates](https://www.searates.com/container/tracking) — Provides tools for tracking containers, calculating shipping rates, and managing logistics operations.
- [Shipit Container Tracking](https://www.shipit.co.uk/container-tracking.htm) — Offers container tracking services for international removals and shipping from the UK.
- [ShipmentLink Cargo Tracking](https://www.shipmentlink.com/servlet/TDB1_CargoTracking.do) — Allows users to track cargo shipments using container, booking, or bill of lading numbers.
- [Track-trace](https://www.track-trace.com/) — Provides tracking services for various carriers, including air cargo, postal, and container shipments.
