# Digital Storytelling

> 24 tools — [osinthandbook.com/digital-storytelling](https://www.osinthandbook.com/digital-storytelling)

- [Adobe Express](https://spark.adobe.com/) — All-in-one creative suite for quick social graphics, web pages, and video stories.
- [Animatron](https://www.animatron.com) — Browser-based animation studio for vector animations, HTML5 banners, and explainer videos.
- [Animoto](https://animoto.com) — Online video slideshow maker with music and themed templates.
- [Atavist](https://magazine.atavist.com/) — Online magazine publishing platform for long-form storytelling with rich media.
- [Juxtapose](https://juxtapose.knightlab.com) — Free browser tool for comparing two images with an interactive swiping slider.
- [Klynt](https://www.klynt.net/) — Interactive editing and publishing application allowing mixed-media narratives with texts, images, audio, video, hyperlinks, visual storyboarding, and more.
- [MakeBeliefsComix](https://makebeliefscomix.com/) — Web-based comic strip generator, allowing users to create and customise characters and scenes.
- [Neatline](https://neatline.org/) — Tool for building narrative-driven multimedia timelines and collections with mapping and media features.
- [Odyssey](https://cartodb.github.io/odyssey.js/) — JavaScript library for building multimedia, scroll-driven stories.
- [Pageflow](https://www.pageflow.io) — Web-based multimedia storytelling tool enabling intuitive inline editing, automated optimisation of images / audio / video (including 360°), and fast publishing.
- [Racontr](https://racontr.com) — A collaborative platform for the creation and distribution of interactive projects.
- [RaptMedia](http://www.raptmedia.com) — Platform for creating interactive videos.
- [RawShorts](https://www.rawshorts.com/) — A platform to instantly create animated videos from text.
- [Shorthand](https://shorthand.com) — Visually rich no-code storytelling platform for crafting feature-style articles, digital magazines, reports, and internal documents.\n
- [Soundcite](http://soundcite.knightlab.com) — Tool to add audio commentary directly within text, e.g., annotated quotes in articles.
- [sStory](https://ejfox.github.io/sStory/) — Open source interactive storytelling prototype for branching, media-rich stories.
- [Steller](https://steller.co) — Mobile storytelling app for sharing visual narratives combining photos, text, and maps.
- [StoryMap](https://storymap.knightlab.com) — Tool for creating interactive maps linked with text, images, and media.
- [StoryMaps](https://storymaps.arcgis.com/en) — Web platform for building immersive, map-based narratives with ArcGIS integration.
- [Sway](https://sway.com) — A Microsoft platform to create visually striking newsletters, presentations and documentation.
- [Thinglink](https://www.thinglink.com) — Platform for enriching visual media (images, videos, 360° media, 3D models) with interactive hotspots and analytics.
- [Tripline](https://www.tripline.net/) — Tool for creating and sharing visually driven travel maps that transition from trip planning to journal storytelling.
- [VideoScribe](http://www.videoscribe.co) — Hand-drawn whiteboard-style animation creator for educational and presentation purposes.
- [Wevideo](https://www.wevideo.com) — Cloud-based video editing platform featuring collaborative timelines, stock media, transitions, and direct sharing.
