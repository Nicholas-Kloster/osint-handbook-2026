# Wifi Search

- [Wifi Finder Connect Internet Access](https://itunes.apple.com/us/app/wifi-finder-free-internet/id1011519183?mt=8&ign-mpt=uo%3D8) — App for finding free internet and Wi-Fi hotspots.
- [Wifi Map](https://www.wifimap.io) — App and map for locating free Wi-Fi hotspots.
- [Wifi Places](https://wifiplaces.evidweb.com) — Directory of public Wi-Fi locations by category and region.
- [Wifi Space](https://wifispc.com) — App for finding free internet and Wi-Fi hotspots.
- [WifiCafeSpots](https://www.wificafespots.com/) — Directory of cafes and restaurants with free Wi-Fi.
- [WifiFreeSpot](https://www.wififreespot.com/) — Directory of hotels, cafes, and other public Wi-Fi spots.
- [Wigle](https://wigle.net) — Database and map of wireless networks and geolocation data.

- [WIreTapper](https://github.com/h9zdev/WireTapper) — WireTapper is a wireless OSINT tool that passively detects and maps Wi-Fi, Bluetooth, CCTV cameras, vehicles, headphones, TVs, IoT devices, and cell towers, turning nearby radio signals into clear situational intelligence.
- [Xfinity Wifi](https://wifi.xfinity.com) — Map and access tool for Xfinity public Wi-Fi hotspots.
