# Working With Emojis

> 8 tools — [osinthandbook.com/working-with-emojis](https://www.osinthandbook.com/working-with-emojis)

- [Emoji Translate](https://meowni.ca/emoji-translate/) — Real‑time emoji translator; type English sentences and it instantly replaces words with emojis.
- [Emoji Translator](https://lingojam.com/%F0%9F%90%8CEmojiTranslator%F0%9F%8E%B2) — Online tool that converts input text into a string of contextually relevant emojis using keyword mapping.
- [Emojipedia](https://emojipedia.org) — Comprehensive emoji reference.
- [EmojiTracker](https://emojitracker.com) — A tool tracking all the emojis copied from Emojipedia and GetEmoji in real time.
- [Fast Emoji](https://www.fastemoji.com) — Web app and browser extension showing trending emojis and providing quick access / search functionality.
- [GetEmoji](https://getemoji.com) — Copy and paste emojis for Twitter, Facebook, Slack, Instagram, Snapchat, Slack, GitHub, Instagram, WhatsApp and more.
- [iEmoji](https://www.iemoji.com) — Emoji encyclopedia offering copy‑paste emoji art, descriptions, and platform variants.
- [Super Emoji Translator](http://superemojitranslator.com) — Simple “Super Emoji Translator” that swaps key English words with emojis while preserving other text.
