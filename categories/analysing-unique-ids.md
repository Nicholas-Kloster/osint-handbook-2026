# Analysing Unique Ids

> 9 tools — [osinthandbook.com/analysing-unique-ids](https://www.osinthandbook.com/analysing-unique-ids)

- [Analyze ID](https://analyzeid.com) — Reverse-lookup tool for identifiers like Google Analytics, Adsense, Amazon affiliate codes, emails, IPs, and more.
- [DomainIQ: Reverse Analytics](https://www.domainiq.com/reverse_analytics) — Domain analytics platform offering insights on reverse Google analytics, domain ownership, hosting history, and the like.
- [Hacker Target: Reverse Analytics Search](https://hackertarget.com/reverse-analytics-search) — Online tool to perform reverse-Google-Analytics or Adsense ID searches to find associated web assets.
- [Moonsearch: Analytics](https://moonsearch.com/analytics) — Analytics exploration / search tool for discovering web properties via tracking identifiers.
- [OSINT.sh: Reverse Google Adsense](https://osint.sh/adsense) — Find domains with the same Google Adsense ID
- [Reverse Analytics](https://dnslytics.com/reverse-analytics) — Reverse analytics / Google tag platform that findd domains sharing the same Analytics ID or Google Tag.
- [Reverse Google Analytics](https://osint.sh/analytics) — Find domains with the same Google Analytics ID.
- [Synapsint](https://synapsint.com) — A free service that aggregates data from multiple intelligence sources, search engines, datasets, and other resources to provide comprehensive cybersecurity insights.
- [wayback-google-analytics](https://github.com/bellingcat/wayback-google-analytics) — A lightweight tool for scraping current and historic Google Analytics data.
