# Linux Distributions

> 22 tools — [osinthandbook.com/linux-distributions](https://www.osinthandbook.com/linux-distributions)

- [Arch Strike](https://archstrike.org/) — An Arch Linux repository for security professionals and enthusiasts.
- [BackBox](https://www.backbox.org) — Ubuntu-based distro with a curated ethical hacking toolkit.
- [Black Arch](https://www.blackarch.org/) — An Arch Linux-based penetration testing distribution for penetration testers and security researchers.
- [CSI Linux](https://csilinux.com/) — Linux built for cybersecurity professionals providing integrated digital forensics, OSINT, and incident-response utilities.
- [Fedora](https://getfedora.org) — Community-driven, polished desktop OS based on Fedora / Red Hat, ideal for developers and general users.
- [Kali Linx](https://www.kali.org/) — Debian-based platform tailored for penetration testing, with extensive security tools and strong documentation support.
- [Kodachi](https://www.digi77.com/linux-kodachi/) — Anti-forensic, privacy-focused live distro based on Xubuntu.
- [Mint](https://linuxmint.com) — User-friendly desktop distro based on Ubuntu and Debian, emphasising ease-of-use and out-of-the-box functionality.
- [OSINTko](https://github.com/LinaYorda/OSINTko) — OSINTk.o is a customised Kali Linux-based ISO image with pre-installed packages and scripts.
- [OSINTux](https://www.osintux.org/) — Security and OSINT Linux distro (OSINTUX) designed for open-source intelligence tasks.
- [Parrot](https://www.parrotsec.org) — Debian-based security distro providing tools for pentesting, forensics, and privacy with beginner-friendly interface.
- [Pentoo](https://www.pentoo.ch/) — Gentoo-based live distro and overlay for penetration testing, with hardened kernel and GPGPU cracking support.
- [Pure OS](https://pureos.net) — A user-friendly Linux distribution focusing on privacy and security.
- [Qubes](\" preventing malware from spreading between compartments.") — Security-oriented desktop OS using Xen virtualisation to isolate tasks into \qubes
- [Septor](https://septor.sourceforge.io/) — Debian-based distro with KDE Plasma and Tor integration for anonymous internet browsing.
- [SIFT Workstation](https://www.sans.org/tools/sift-workstation/) — A free and open-source forensic and incident-response toolkit for detailed digital forensic analysis.
- [Tails](https://tails.boum.org) — A Debian-based OS that boots from USB or DVD, routes all traffic through Tor, and leaves no trace on the host machine.
- [TinHat](https://sourceforge.net/projects/tinhat/) — Hardened Gentoo-based distro running entirely in RAM, offering enhanced security via ephemeral execution.
- [tlosint-live](https://github.com/tracelabs/tlosint-live) — Trace Labs OSINT Linux Distribution based on Kali.
- [Tsurugi](https://tsurugi-linux.org) — Customised Linux distro tailored for digital forensics, malware analysis, and OSINT investigations.
- [Ubuntu](https://ubuntu.com) — Popular, polished Ubuntu distro covering desktop, server, cloud, and IoT use cases with regular LTS releases.
- [whonix](https://www.whonix.org) — Privacy-oriented, virtual-machine-based OS separating network and app functions for enhanced anonymity.
