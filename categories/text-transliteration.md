# Text Transliteration

> 7 tools — [osinthandbook.com/text-transliteration](https://www.osinthandbook.com/text-transliteration)

- [Lexilogos Arabic Transliteration](https://www.lexilogos.com/keyboard/arabic_latin.htm) — Arabic transliteration tool.
- [Lexilogos Cyrillic-Latin Conversion](https://www.lexilogos.com/keyboard/russian_conversion.htm) — Cyrillic-Latin conversion tool.
- [My Languages](http://mylanguages.org/arabic_romanization.php) — Arabic to Latin converter.
- [Text Magic](https://www.textmagic.com/free-tools/text-transliteration-tool) — Transliterate text message characters from non-Latin to Latin.
- [Transliterating Arabic to English in One Step](https://stevemorse.org/arabic/ara2eng.html) — Arabic to English transliteration tool.
- [Typeit](https://www.typeit.org) — Type accent marks, diacritics and other characters online.
- [Universal Cyrylic Decoder](https://2cyr.com/decode/?lang=en) — Universal online tool for decoding and recovering text into the Cyrillic alphabet.
