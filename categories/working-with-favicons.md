# Working With Favicons

> 5 tools — [osinthandbook.com/working-with-favicons](https://www.osinthandbook.com/working-with-favicons)

- [Fav-up](https://github.com/pielco11/fav-up) — Lookups for real IP starting from the favicon icon and using Shodan.
- [Favi Hash](https://www.favihash.com/) — Get the hash of a favicon to identify websites using the same.
- [favihunter](https://github.com/eremit4/favihunter) — Discover and monitor internet assets using favicon hashes across search engines.
- [favihunter](https://github.com/sharsil/favicorn) — An all-sources tool to search websites by favicons.
- [Search Result Previews](https://chromewebstore.google.com/detail/search-result-previews/cedcejfiniojnlhlfhcppenochinijfo) — Display thumbnails and favicons next to search results for quick page previews.
