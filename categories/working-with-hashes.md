# Working With Hashes

> 13 tools — [osinthandbook.com/working-with-hashes](https://www.osinthandbook.com/working-with-hashes)

- [_IntelligenceX Reverse Hash Lookup](https://intelx.io/tools?tab=hash) — OSINT reverse-hash lookup tool (MD5, SHA1, SHA256, SHA512),
- [CrackStation](https://crackstation.net) — High-capacity hash-cracking service using extensive lookup tables to reverse MD5, SHA1, SHA2, WPA and more.
- [DeepFind.me: File Hash Checker](https://www.deepfind.me/tools/file-analysis/file-hash-checker) — Verify file integrity or check against known malicious hashes.
- [Hashes](https://hashes.com/en/decrypt/hash) — Free online platform for decrypting a variety of hashes.
- [HashID](https://mattw.io/hashID/) — Browser-based tool that identifies the hash type (e.g., MD5, SHA variants) from user input.
- [Hashmob](https://hashmob.net/search) — Community-powered hash search where users can query hashes to see if their plaintext has already been discovered by others.
- [M5decrypt](https://md5decrypt.net/en/#answer) — Online service for encrypting and decrypting (reverse lookup) MD5 hashes.
- [Mobile Fish Hash Generator](https://www.mobilefish.com/services/hash_generator/hash_generator.php) — Web-based generator for common hash algorithms.
- [Online Hash Crack](https://www.onlinehashcrack.com/hash-identification.php) — Hash identification interface using a large reverse-lookup database to determine the algorithm used.
- [OSINT.sh: MD5](https://osint.sh/md5) — Search for information on an MD5 hash.
- [Sha-256 Hash Calculator](https://xorbin.com/tools/sha256-hash-calculator) — Online SHA-256 hash calculator.
- [Sha1](http://www.sha1-online.com) — Web page offering SHA-1 and additional hash generation tools.
- [shodan-hash](https://book.shodan.io/command-line-tools/shodan-hash) — A command-line to calculate simple hashes of a website, favicon, local file or text. It can be used freely without a Shodan account.
