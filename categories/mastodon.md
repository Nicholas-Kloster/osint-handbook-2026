# Mastodon

> 10 tools — [osinthandbook.com/mastodon](https://www.osinthandbook.com/mastodon)

- [Fediverse Explorer](https://fediverse.info/explore/people) — A directory of Fediverse users.
- [Followgraph on Mastodon](https://followgraph.vercel.app/) — Analyzes follows and suggests new accounts by examining mutual networks.
- [IMAGSTODON](https://seintpl.github.io/imagstodon/) — Searches across multiple Mastodon instances to locate user profiles and avatars.
- [Instances Social](https://instances.social/) — Explorer for finding and comparing Mastodon instances based on user preferences.
- [MASTO](https://github.com/C3n7ral051nt4g3ncy/Masto) — An OSINT tool written in Python to gather intelligence on Mastodon users and instances.
- [MastoGizmos](https://mastogizmos.com/) — A collection of one-page JavaScript apps offering new ways to search and explore Mastodon's content.
- [Movetodon](https://movetodon.org/) — Helps users migrate from Twitter to Mastodon by finding friends with Mastodon accounts.
- [OSoMe Mastodon Search](https://osome.iu.edu/tools/mastodon/) — Search interface to retrieve Mastodon statuses, accounts, hashtags, metadata, and instances.
- [SNScrape](https://github.com/JustAnotherArchivist/snscrape) — A simple Python script that scrapes user profiles, hashtags, and search results from a variety of social media platforms.
- [Trunk](https://communitywiki.org/trunk/) — Mass-follow a bunch of people in order to get started with Mastodon or any other platform on the Fediverse.
