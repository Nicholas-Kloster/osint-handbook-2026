# Private And Local Llms

> 8 tools — [osinthandbook.com/private-and-local-llms](https://www.osinthandbook.com/private-and-local-llms)

- [Datasaur](https://datasaur.ai) — An end-to-end data labeling and AI services platform for NLP model training and deployment.
- [GPT4All](https://www.nomic.ai/gpt4all) — A private, local AI chatbot for Mac OS, Windows, and Linux.
- [Jan](https://jan.ai) — An open source ChatGPT-alternative that runs 100% offline. Available for Mac OS, Windows, and Linux.
- [llama.cpp](https://github.com/ggml-org/llama.cpp) — An LLM inference in C/C++.
- [llamafile](https://github.com/Mozilla-Ocho/llamafile) — Distribute and run LLMs with a single file.
- [LM Studio](https://lmstudio.ai) — A downloadable desktop app that lets you run open-source LLMs like GPT-OSS, Qwen, Gemma, and DeepSeek locally on your computer.
- [NextChat](https://nextchat.club) — An open-source AI chat interface.
- [Ollama](https://ollama.com) — An open-source tool for running large language models locally on your own machine.
