# Social Media Search

> 7 tools — [osinthandbook.com/social-media-search](https://www.osinthandbook.com/social-media-search)

- [buzzglobe](https://buzzglobe.com) — A tool to search different social media platforms without logging in.
- [Evolve Social Search](https://bonobos.github.io/evolve-social-search/) — A simple tool that generates search URLs for various social media platforms.
- [PB CSV](https://cse.google.com/cse?cx=029ffbc44aa3946cb) — A custom search engine that searches within more than 17 different social media platforms.
- [Social Search Engine](https://www.socialsearchengine.org/) — A custom search engine that searches within multiple social media platforms including Facebook, Twitter, and more.
- [Social Searcher](https://www.social-searcher.com/) — A free social media search engine.
- [UVRX](http://www.uvrx.com/social.html) — A custom search engine that searches within multiple social media platforms.
- [Waybien](https://waybien.com) — Easily find channels, groups, and servers on Facebook, Telegram, Discord, and WhatsApp.
