# Technology Lookup

> 9 tools — [osinthandbook.com/technology-lookup](https://www.osinthandbook.com/technology-lookup)

- [BuiltWith](https://builtwith.com/) — A popular technology lookup platform.
- [NerdyData](https://www.nerdydata.com/) — A search engine for finding source code and technologies across the web.
- [Netcraft Search DNS](https://searchdns.netcraft.com) — Tool for DNS and hosting history lookup including site technology and uptime details.
- [Netcraft Site Report](http://toolbar.netcraft.com/site_report) — Find out the infrastructure and technologies used by any site.
- [OSINT.sh: Technology Lookup](https://osint.sh/stack) — Technology stack checker tool. Check out the technologies used on any website.\n\n
- [urlscan](https://urlscan.io) — A free service to scan and analyse websites.
- [W3techs Sites](https://w3techs.com/sites) — Find out, which technologies such as CMS, programming language, web server and hosting provider a particular website is using.
- [Wappalyzer](https://wappalyzer.com) — Web technology profiler.
- [What Wordpress Theme Is That](https://whatwpthemeisthat.com) — A tool that identifies a Wordpress theme used by a given website.
