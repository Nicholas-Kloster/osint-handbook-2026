# Font Identification

> 6 tools — [osinthandbook.com/font-identification](https://www.osinthandbook.com/font-identification)

- [Font Spring](https://www.fontspring.com/matcherator) — A font recognition tool that identifies fonts from uploaded images or text.
- [Font Squirrel](https://www.fontsquirrel.com/matcherator) — A font recognition tool that extracts letter shapes from user images and matches to fonts.
- [Identifont](http://www.identifont.com/) — Directory and expert-system tool that identifies fonts by asking a series of questions about appearance and offers matches by name, designer, similarity, or symbol.
- [LikeFont](https://en.likefont.com/) — Free online image-based font recognition platform.
- [What Font Is](https://www.whatfontis.com/) — AI-powered font finder that matches fonts from uploaded images against a huge database of free and commercial fonts, showing many similar options.
- [What The Font](https://www.myfonts.com/pages/whatthefont) — A font recognition tool that identifies fonts from uploaded images or text.
