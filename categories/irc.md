# Irc

> 4 tools — [osinthandbook.com/irc](https://www.osinthandbook.com/irc)

- [IRC Channel Search](http://www.efnet.org/?module=channels) — Provides a searchable index of IRC channels across the EFnet network.
- [IRC Chat Rooms](https://netsplit.de/channels/) — Displays live statistics and lists of active IRC channels by user count.
- [sunxdcc](https://sunxdcc.com) — Enables users to search for and browse XDCC bot listings on supported IRC networks.
- [XDCC Search Engine](https://www.xdcc.eu) — Offers a directory of XDCC file search results from various IRC networks.
