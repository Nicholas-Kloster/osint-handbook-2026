# Chrome Extensions

> 15 tools — [osinthandbook.com/chrome-extensions](https://www.osinthandbook.com/chrome-extensions)

- [Beyond 10](https://chromewebstore.google.com/detail/beyondten-show-100-google/dbkkfpfndpelicmjdpfacabhjffgoacl) — View up to 100 Google search results per page.
- [Context Menu Search](https://chrome.google.com/webstore/detail/context-menu-search/ocpcmghnefmdhljkoiapafejjohldoga) — Customize your context menu to search selected text using various search engines.
- [Google Dork Builder](https://addons.mozilla.org/ru/firefox/addon/google-dork-builder/) — Firefox extension for building and testing Google dork queries.
- [Instant Preview](https://chrome.google.com/webstore/detail/search-preview/majhgbekihmliceijipbdccgicepmmei) — Add previews and rankings to search results for better insight before clicking.
- [Multi Find](https://chromewebstore.google.com/detail/multi-find-search-and-hig/dffaiikpbncahnghlfnkhagffaemhgfo) — Search and highlight multiple keywords simultaneously on a webpage.
- [Remove Breadcrumbs](https://chromewebstore.google.com/detail/remove-breadcrumbs/banhponphmmpnpogmfaahcgkgbgkcoka) — Replace breadcrumb paths with full URLs in search results for clearer navigation.
- [Save Search Results](https://chromewebstore.google.com/detail/save-search-results/fppgodjcdekgbhbipabkkdhbmnahbhnj) — Save URLs from Google search results with a single click for easy reference.
- [Search & Link Preview](https://chromewebstore.google.com/detail/search-link-preview/mmmfofondapflhgbdidadejnechhjocm) — Preview links and perform searches within the same page to enhance browsing efficiency.
- [Search All](https://chrome.google.com/webstore/detail/search-all/kpdkbemdpepjjppbfgeapjienologapa) — Perform searches across multiple search engines simultaneously for comprehensive results.
- [Search All Tabs](https://chromewebstore.google.com/detail/search-all-tabs/ndfnfhnlgoocpbbjjjfmiojdfcgcfomn) — Search across all open tabs to quickly locate specific content or titles.
- [Search Result Previews](https://chromewebstore.google.com/detail/search-result-previews/cedcejfiniojnlhlfhcppenochinijfo) — Display thumbnails and favicons next to search results for quick page previews.
- [Search the current site](https://chrome.google.com/webstore/detail/search-the-current-site/jliolpcnkmolaaecncdfeofombdekjcp) — Search within the current website to find specific content without leaving the page.
- [Selection Search](https://chrome.google.com/webstore/detail/selection-search/gipnlpdeieaidmmeaichnddnmjmcakoe) — Search selected text using various search engines directly from the context menu.
- [Simple = Select + Search](https://chromewebstore.google.com/detail/simple-select-+-search/aagminaekdpcfimcbhknlgjmpnnnmooo?hl=en) — Select text and search it using your preferred search engine directly from the context menu.
- [uBlacklist](https://chromewebstore.google.com/detail/ublacklist/pncfbmialoiaghdehhbnbhkkgmjanfhe) — Block specific sites from appearing in your search results.
